# プロジェクト概要

## 目的
組織図などの階層データを視覚的に編集できるインタラクティブなツリー構造エディタ

## 主要機能
- 階層データのインポート/エクスポート（`親 > 子 > 孫` 形式）
- ノードの追加・削除・移動
- ドラッグ&ドロップによる移動（before/after/inside）
- 同一階層での重複名検証
- ノードの展開/折りたたみ
- トースト通知によるフィードバック

## バリデーション
- ルートノードは1つまで
- 同一階層でのノード名重複を防止（異なる階層では同名可）
- 親ノードをその子孫に移動不可
- 空白のみのノード名は不可（前後の空白は自動トリム）

## デプロイ
- GitHub Pagesに自動デプロイ
- `master`ブランチへのpushで自動テスト→ビルド→デプロイ
- 公開URL: https://pyyoshi.github.io/demo-interactive-tree-structure-editor/
